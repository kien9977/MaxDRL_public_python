# MaxDRL_public_python

# Step to run
+ Step 1: install Python 3.x
+ Step 2: using pip to install tqdm and requests
```
python -m pip install tqdm
python -m pip install requests
```
+ Step 3: Running this python by sample command
```
python main.py
```
